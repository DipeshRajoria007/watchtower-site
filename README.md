# Watchtower Site

This repo is the front porch for Watchtower, now built on Next.js App Router instead of Vite.

The site is intentionally server-rendered by default, ships self-hosted fonts through `next/font`, and keeps the landing page free of unnecessary client-side React work so the Core Web Vitals stay strong.

## What This Page Is Selling

Watchtower is a macOS-native product for teams that want autonomous developer workflows without surrendering visibility.

The site tells that story in a more cinematic way than the app repo does:

- Slack mention in
- the right workflow wakes up
- action happens
- traces, proof, and learning stay visible

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run start
```
